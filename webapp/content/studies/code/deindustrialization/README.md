# Deindustrialization: The Long Decline of U.S. Manufacturing, 1997-2024

Tracks manufacturing's shrinking share of the U.S. economy across 28 years
using BEA input-output benchmarks.

## Run

```bash
pip install -r requirements.txt
python analysis.py
```

Outputs: `outputs/deindustrialization.csv`, `outputs/fig_deindustrialization_trend.json`

## Data

| File | Description |
|---|---|
| `data/deindustrialization.csv` | Year, manufacturing VA share of GDP, manufacturing gross output share (1997-2024) |

Exported from `Outputs/Data/deindustrialization_1997_2024.xlsx` (BEA-verified pipeline).
