# Wassily - Input-Output Tables Analysis Tool

**Purpose:** Comprehensive analysis tool for U.S. and international Input-Output tables
**Status:** Initial Setup
**Target Audience:** Researchers, economists, and analysts studying economic structure and interdependencies
**Named After:** Wassily Leontief (1906-1999), Nobel laureate and pioneer of input-output analysis

---

## Project Overview

Wassily is a Python-based analytical tool designed to:
- Open and analyze every vintage of U.S. input-output tables (BEA)
- Process international input-output tables (OECD, WIOD, etc.)
- Perform comprehensive structural analysis of economic interdependencies
- Generate insights on sectoral linkages, multipliers, and economic structure

## About Wassily Leontief

Wassily Leontief developed input-output analysis to understand how different sectors of an economy interact. His work provides a systematic framework for analyzing the structure of production, consumption, and trade. This tool continues his legacy by making I-O analysis accessible and comprehensive.
