## Executive Summary

This guide provides a comprehensive inventory of Input-Output (I-O) data sources available for economic analysis, ranging from national accounts to global multi-regional databases. The resources are organized by geographic scope, methodological approach, and analytical capabilities to support the expansion of the Leontief.io platform from U.S.-centric analysis to global economic structure analysis.

**Primary Finding:** The global I-O data ecosystem is rich and diverse, with multiple overlapping sources that provide complementary coverage of different time periods, geographic regions, and analytical dimensions.

**Key Recommendation:** Prioritize integration in the order of: U.S. annual extension → European core databases → Global multi-regional → Environmental extensions.

## North American Sources

### United States - BEA (Bureau of Economic Analysis)

#### Historical Benchmark Tables
- **Coverage:** 1997, 2002, 2007 comprehensive benchmarks
- **Industry Detail:** 400-500 detailed industries
- **Data Products:**
  - Make and Use tables (detailed and summary)
  - Requirements tables (direct and total)
  - Final demand components
  - Employment and compensation data
- **Methodology:** Full economic census, complete enumeration
- **Access:** BEA website, API, bulk downloads
- **Status:** Completed integration in Leontief.io v1.0

#### Annual 71-Category Series
- **Coverage:** 1997-present
- **Industry Detail:** 71 aggregated categories
- **Frequency:** Annual releases
- **Methodology:** Sample surveys + administrative data
- **Quality:** Strong correlation with benchmarks (r > 0.95)
- **Status:** Research completed, integration pending

#### Regional I-O Accounts
- **Coverage:** States, metropolitan areas
- **Industry Detail:** Variable (typically 20-50 industries)
- **Data Products:** Regional multipliers, impact estimates
- **Integration Potential:** Medium (classification alignment challenges)

### Canada - Statistics Canada

#### Canadian Input-Output Tables
- **Coverage:** 1961-present (annual)
- **Industry Detail:** 234 industries (detailed)
- **Data Products:**
  - Symmetric I-O tables
  - Use and make tables
  - Final demand categories
  - Import/export matrices
- **Methodology:** Supply and use framework, balanced
- **Access:** Statistics Canada website, CANSIM database
- **Integration Priority:** Medium (North American trade analysis)

### Mexico - INEGI

#### Mexican Input-Output Tables
- **Coverage:** 2003, 2008, 2013 benchmarks
- **Industry Detail:** 259 industries (2013)
- **Data Products:**
  - Use and make tables
  - Import content of exports
  - Regional breakdowns
- **Methodology:** NAICS-based classification
- **Access:** INEGI website, specialized requests
- **Integration Priority:** Medium (NAFTA/USMCA analysis)

## European Sources

### WIOD (World Input-Output Database)

#### 2016 Release
- **Coverage:** 43 countries, 2000-2014
- **Industry Detail:** 56 industries (ISIC Rev. 3)
- **Data Products:**
  - World I-O tables (WIOT)
  - National I-O tables (NIOT)
  - International supply and use tables
  - Socio-economic accounts
  - Environmental accounts (optional)
- **Methodology:** Consistent international classification
- **Access:** WIOD website, downloadable Excel/CSV
- **Status:** Ready for integration

#### Analytical Capabilities
- Multi-country analysis
- Global value chain analysis
- Trade in value-added (TiVA) analysis
- Employment and skill analysis
- Environmental extension integration

### OECD ICIO (Inter-Country Input-Output) Tables

#### Annual Series
- **Coverage:** 64+ countries, 1995-present
- **Industry Detail:** 36 industries (ISIC Rev. 4)
- **Frequency:** Annual updates
- **Data Products:**
  - ICIO tables
  - TiVA indicators
  - Bilateral trade flows
  - Global value chain metrics
- **Methodology:** Harmonized with national accounts
- **Access:** OECD Statistics database
- **Integration Priority:** High (contemporary coverage)

### European Commission - ESA I-O Tables

#### ESA 2010 Compatible Tables
- **Coverage:** EU-27 + candidate countries
- **Time Period:** 1995-present
- **Industry Detail:** 64 industries (NACE Rev. 2)
- **Data Products:**
  - Symmetric I-O tables
  - Supply and use tables
  - Import/export matrices
- **Methodology:** ESA 2010 standards
- **Access:** Eurostat database
- **Integration Priority:** Medium (European focus)

## Global Multi-Regional Databases

### EORA MRIO (Multi-Regional Input-Output)

#### 26 Country Version
- **Coverage:** 26 major economies, 1990-2018
- **Industry Detail:** 26 sectors (aggregated)
- **Resolution:** Country-level
- **Methodology:** Consistent global framework
- **Access:** EORA website (license-based)

#### 190 Country Version
- **Coverage:** 190 countries, 1990-2018
- **Industry Detail:** Variable (typically 15-25 sectors)
- **Resolution:** Comprehensive global coverage
- **Methodology:** Trade balance reconciliation
- **Access:** Commercial license required
- **Integration Priority:** High (comprehensive global coverage)

### EXIOBASE 3

#### Full Environmental Extension
- **Coverage:** 44 countries + 5 rest-of-world regions
- **Time Period:** 1995-2011 (base year 2007)
- **Industry Detail:** 129 industries (product classification)
- **Resolution:** Country-by-country
- **Data Products:**
  - Detailed MRIO tables
  - Environmental extensions (CO2, materials, water)
  - Satellite accounts
  - Multi-pressure analysis
- **Methodology:** Environmental satellite accounting
- **Access:** Open access (registration required)
- **Integration Priority:** High (environmental analysis)

### GTAP (Global Trade Analysis Project)

#### GTAP 10 Database
- **Coverage:** 141 countries/regions
- **Base Year:** 2014 (continuous updates)
- **Industry Detail:** 65 sectors
- **Data Products:**
  - Bilateral trade flows
  - Production and consumption
  - Protection data (tariffs, subsidies)
  - Energy data
- **Methodology:** Partial equilibrium framework
- **Access:** Subscription-based
- **Integration Priority:** Medium (trade policy analysis)

## Methodological Comparisons

### Classification Systems

| Database | Classification | Industries | Update Frequency |
|----------|----------------|------------|------------------|
| BEA US | NAICS | 71 (annual) / 400+ (benchmark) | Annual/5-year |
| WIOD | ISIC Rev. 3 | 56 | Historical |
| OECD ICIO | ISIC Rev. 4 | 36 | Annual |
| EXIOBASE 3 | Custom | 129 | Historical |
| EORA | Custom | 15-25 | Historical |

### Balance Methods

1. **Supply-Use Framework (SUT)**
   - Used by: BEA, Statistics Canada, Eurostat
   - Advantages: Economic consistency, detailed commodity flows
   - Challenges: Reconciliation requirements

2. **Direct Symmetric Tables**
   - Used by: WIOD, EORA (some versions)
   - Advantages: Simpler structure, direct coefficients
   - Challenges: Less economic detail

3. **Hybrid Approaches**
   - Used by: EXIOBASE, some OECD products
   - Advantages: Balance of detail and consistency
   - Challenges: Methodological complexity

### Quality Dimensions

**Statistical Reliability:**
- Benchmark > Annual > Model-based estimates
- Census > Survey > Administrative data

**Temporal Consistency:**
- Consistent classification > Reconciled time series
- Regular updates > Historical revisions

**Geographic Coverage:**
- Global comprehensive > Regional detailed > National specific

## Software and Tools

### Standalone Software

#### IMPLAN (Commercial)
- **Function:** Economic impact analysis
- **Data Source:** Proprietary U.S. databases
- **Capabilities:** Multi-region analysis, construction modeling
- **Cost:** High (commercial license)

#### RAS (Regional Analysis System)
- **Function:** Regional economic modeling
- **Data Source:** User-provided I-O tables
- **Capabilities:** Regionalization, impact analysis
- **Cost:** Open source

### Statistical Packages

#### R - ioanalysis Package
- **Function:** I-O analysis toolkit
- **Features:** Leontief inverse, multipliers, decomposition
- **Integration:** R ecosystem compatibility
- **Cost:** Open source

#### Python - PyMARIO
- **Function:** Multi-regional I-O analysis
- **Features:** Scenario analysis, environmental extensions
- **Integration:** Python ecosystem
- **Cost:** Open source

#### MATLAB I-O Toolbox
- **Function:** Academic research toolkit
- **Features:** Advanced analytical methods
- **Integration:** MATLAB environment
- **Cost:** Academic license

### Data Processing Tools

#### Custom Scripts
- **Languages:** Python, R, MATLAB
- **Functions:** Data cleaning, classification conversion, balance reconciliation
- **Requirements:** Programming expertise

#### Excel/Access Templates
- **Function:** Basic I-O analysis
- **Limitations:** Size constraints, manual processes
- **Suitability:** Small-scale analysis
