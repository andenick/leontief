# Leontief Platform

Interactive web application for exploring Input-Output analysis results across three methodological approaches.

## Features

### 6 Interactive Tabs

1. **📊 Methodology Comparison** - Side-by-side comparison of all three methods
2. **🔍 Industry Explorer** - Browse and search industry-level multipliers
3. **🧮 Impact Calculator** - Estimate economic impacts of direct investments
4. **📈 Sector Analysis** - Visualize patterns and relationships in multipliers
5. **⚙️ Special Industries** - Deep dive into BEA's 10 redefinition industries
6. **💾 Data Export** - Download results in various formats

## Installation

### Prerequisites

- Python 3.8 or higher
- pip package manager

### Setup

1. Navigate to the apps directory:
```bash
cd <project-root>/Technical/apps
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Running the Platform

From the apps directory, run:

```bash
streamlit run wassily_platform.py
```

The platform will open in your default web browser at `http://localhost:8501`

## Data Requirements

The platform loads data from the following files (automatically included):

- `Output/Data/industry_by_industry_2002.pkl` (Method 1 results)
- `Output/Data/method2_bea_direct_2002.pkl` (Method 2 results)
- `Output/Data/method3_reverse_engineered_2002.pkl` (Method 3 results)
- `Output/Data/[2025.10.10] comprehensive_methods_comparison.xlsx` (Comparison data)

All files are generated by the analysis scripts and should be in place.

## Usage Guide

### Methodology Comparison Tab
- View summary statistics for all three methods
- Explore distribution comparisons
- Understand the 1.96× relationship

### Industry Explorer Tab
- Search for specific industries by code
- Sort by different metrics
- View detailed comparisons for individual industries

### Impact Calculator Tab
- Select an industry
- Enter a direct investment amount
- See total economic impact under all three methods
- Understand methodological sensitivity

### Sector Analysis Tab
- Visualize correlations between methods
- Identify outliers and patterns
- Compare top and bottom performers

### Special Industries Tab
- Explore BEA's 10 special redefinition industries
- Understand S-codes and T-codes
- See how special industries differ from standard classification

### Data Export Tab
- Download multipliers in CSV format
- Export summary statistics
- Access ratio analysis
- Preview data before downloading

## Key Findings

The platform reveals that:

- **BEA multipliers are 1.9645× commodity technology values** (systematic relationship)
- **S00201 (Petroleum)** has the highest multiplier (8.63) in BEA methodology
- **416 common industries** across all three methods
- **10 special industries** in BEA approach not in commodity technology
